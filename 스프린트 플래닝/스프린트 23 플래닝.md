        - PATCH /api/v1/feeds/{feed_id} (피드 수정)

    2. **댓글 관련 API**  
        - GET /api/v1/feeds/{feed_id}/comments (댓글 목록 조회)
        - POST /api/v1/feeds/{feed_id}/comments (댓글 작성)
        - DELETE /api/v1/comments/{comment_id} (댓글 삭제)
    3. **좋아요 및 기타 상호작용 API** 
        - POST /api/v1/feeds/{feed_id}/like (좋아요/취소)
            - requestBody: t/f
        - POST /api/v1/comments/{commentId}/like (좋아요/취소)
            - requestBody: t/f
    4. **마이페이지 관련 API** 
        - GET /api/v1/mypage/feeds (내가 작성한 피드 조회)
